# Samsung Memory Lens

An intelligent photo and video gallery application with voice-controlled search capabilities and AI-powered content analysis.

## 📱 Project Versions

This repository contains two implementations of the Samsung Memory Lens project:

### 🌐 Web Application
**Location**: `/web-app/`
- **Technology**: Node.js, Express, EJS
- **Features**: Web-based photo gallery with AWS Rekognition integration
- **Languages**: JavaScript, EJS, CSS, Python
- **Setup**: See `/web-app/README.md`

### 📱 Mobile Application
**Location**: `/mobile-app/`
- **Technology**: Flutter (iOS & Android)
- **Features**: Native mobile gallery app with voice recording and speech-to-text
- **Setup**: See `/mobile-app/README.md`

## 🚀 Quick Start

Choose your preferred version:

```bash
# For Web App
cd web-app
npm install
npm start

# For Mobile App
cd mobile-app
flutter pub get
flutter run
```

## 🎯 Core Features

- **Smart Gallery**: Browse photos and videos from multiple sources
- **Voice Control**: Record voice commands and convert to text
- **AI Integration**: Content analysis and intelligent search
- **Cross-Platform**: Web browser or native mobile apps
